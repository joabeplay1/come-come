# Come Come SITI WEB 🟡

Jogo Come Come (Pac-Man) multiplayer online construído com React + Canvas.

## 🚀 Como rodar no GitHub Pages

1. Faça fork deste repositório
2. Vá em Settings > Pages
3. Selecione a branch main
4. Acesse: https://seuusuario.github.io/come-come/

## 🎮 Modos de Jogo

- **1 Jogador** — Clássico solo
- **2 Jogadores Local** — P1: WASD | P2: IJKL
- **Multiplayer Online** — Crie uma sala e compartilhe o código

## 🔧 Configuração Firebase

Edite `game.js` e substitua `firebaseConfig` com suas credenciais.

## 📦 Tecnologias

- HTML5 Canvas (física e renderização)
- React (UI)
- Firebase (multiplayer online)
- Tailwind CSS (estilos)
